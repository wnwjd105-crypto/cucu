# 언어 진화 지도 앱 개발 계획

전 세계 언어의 역사적 변화를 시간대별로 시각화하여 일반 사용자가 언어의 흐름과 영향 관계를 직관적으로 이해할 수 있는 인터랙티브 지도 앱을 개발합니다.

## 1단계: 데이터 수집 및 구조화

### 1.1 역사적 시간대 정의
- **기원전 시대**: 500~1000년 단위 (유의미한 언어 변화 기준)
  - 기원전 4000-3000년: 4대 문명 발생기
  - 기원전 3000-2000년: 초기 문명 발전기
  - 기원전 2000-1000년: 청동기 문명
  - 기원전 1000-500년: 철기 문명 전파
  - 기원전 500-0년: 고대 문명 성숙기
- **기원후 시대**: 1세기(100년) 단위
  - 1-100년: 로마 제국 전성기
  - 101-200년: 한나라 성숙기
  - 201-300년: 위진남북조 분열기
  - 301-400년: 서로마 제국 쇠퇴기
  - 401-500년: 게르만 대이동
  - 501-600년: 동로마 제국 보존기
  - 601-700년: 이슬람 제국 확장
  - 701-800년: 당나라 전성기
  - 801-900년: 카롤링거 제국
  - 901-1000년: 비잔틴 제국 쇠퇴
  - 1001-1100년: 송나라 문화
  - 1101-1200년: 십자군 전쟁
  - 1201-1300년: 몽골 제국
  - 1301-1400년: 원나라 통치
  - 1401-1500년: 명나라 건국
  - 1501-1600년: 르네상스
  - 1601-1700년: 청나라 초기
  - 1701-1800년: 계몽주의 시대
  - 1801-1900년: 산업혁명
  - 1901-2000년: 제국주의 시대
  - 2001-현재: 현대 언어 분포

### 1.2 언어 데이터 수집
- **WALS Online (World Atlas of Language Structures)**
  - API 문서 확인 및 데이터 구조 파악
  - 언어 위치, 계통, 특성 데이터 추출
- **Glottolog**
  - 언어 계통도 데이터 확보
  - 언어족 및 계층 구조 데이터 수집
- **위키피디아 데이터**
  - 주요 언어별 상세 정보 URL 수집
  - 역사적 배경, 발생 시점, 지리적 분포 데이터

### 1.3 역사적 국가 경계 데이터
- **Cliopatria 데이터셋 통합**
  - 기원전 3400년부터 2024년까지 전 세계 국가 실제 경계 GeoJSON 다운로드
  - 시대별 국가/제국 경계 폴리곤 데이터 추출
  - 국가명 변경 및 영역 변화 데이터 구조화
- **데이터 포맷 변환**
  - GeoJSON을 시간대별로 분류하여 저장
  - 각 시대별 활성 국가/제국 목록 생성

### 1.4 언어 기원 데이터
- 주요 언어 발생 시점 및 지역 정보
  - 중국어: 기원전 1250년, 황하 유역
  - 일본어: 기원전 300년, 일본 열도
  - 한국어: 기원전 1세기, 한반도
  - 산스크리트어: 기원전 1500년, 인도 북부
  - 라틴어: 기원전 7세기, 이탈리아 반도
  - 그리스어: 기원전 1450년, 그리스
  - 아랍어: 기원후 1세기, 아라비아 반도

### 1.5 언어 영향 관계 데이터
- 언어 간 영향 관계 정의
  - 모어-자어 관계 (parent)
  - 차용어 관계 (borrowing)
  - 인접 언어 영향 (contact)
- 시대별 영향 관계 매핑
- 영향 강도 및 방향성 데이터

## 2단계: 백엔드 개발

### 2.1 데이터베이스 설계
- **언어 테이블**
  - id, name, family, origin_year, origin_location(lat, lng), wikipedia_url
- **시간대 테이블**
  - id, name, start_year, end_year, description
- **언어-시간대 연결 테이블**
  - language_id, period_id, status (emerging, established, dominant, expanding, declining, extinct)
- **영향 관계 테이블**
  - id, from_language_id, to_language_id, period_id, type, strength
- **역사적 경계 테이블**
  - period_id, region_name, geojson_coordinates

### 2.2 API 엔드포인트 구현
- **시간대 데이터**
  - GET /api/time-periods: 모든 시간대 목록
- **언어 데이터**
  - GET /api/languages: 모든 언어 정보
  - GET /api/languages/:id: 특정 언어 상세 정보
- **영향 관계 데이터**
  - GET /api/language-influences: 모든 영향 관계
  - GET /api/language-influences/:period_id: 특정 시대 영향 관계
- **역사적 경계 데이터**
  - GET /api/historical-boundaries: 모든 시대 경계
  - GET /api/historical-boundaries/:period_id: 특정 시대 경계

### 2.3 데이터 파싱 및 로드
- WALS/Glottolog 데이터 파이썬 파서 개발
- Cliopatria GeoJSON 데이터 처리 스크립트
- 데이터베이스 초기화 및 데이터 로드

### 3.3 역사적 경계 레이어
- **GeoJSON 폴리곤 렌더링**
  - Leaflet GeoJSON 레이어 사용
  - 실제 국가 경계 폴리곤 표시
  - 시대별 경계 자동 전환
- **국가명 표시**
  - 폴리곤 중심에 국가명 라벨

### 3.4 언어 분포 렌더링
- **언어 마커**
  - 원형 마커로 언어 위치 표시

    - 정착기: 청록

    - 확장중: 연두
    - 쇠퇴기: 노랑

- **언어 기원 마커**
  - 최초 발생 지점 특별 표시

  - 빨간색 테두리 및 그림자 효과
- **언어 영역 폴리곤**
  - 언어 분포 지역 반투명 폴리곤
  - 시대별 영역 변화 애니메이션

### 3.5 언어 영향 시각화
- **화살표 렌더링**

- **영향 강도**
  - 선 두께로 강도 표현
  - 투명도로 영향력 표현
- **애니메이션**
  - 화살표 흐름 애니메이션

## 4단계: 인터랙티브 기능

### 4.1 클릭 상세 정보
- **언어 클릭**
  - 팝업으로 언어 상세 정보 표시

### 4.4 검색 기능
